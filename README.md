## Installation & Usage

1. Clone this repository:

    ```
    git clone https://github.com/your-username/india-tax-gst-ai-chatbot.git
    ```

2. Open `index.html` in your browser.

3. Start chatting! Examples:

    - `Calculate GST on 5000 for shampoo`
    - `What is income tax on 1200000`
    - `Budget planning for monthly income 40000`
    - `Latest finance ministry GST update`
